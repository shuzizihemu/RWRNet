# 配置环境


| **配置项**       | **要求**                            |
| :--------------- | :---------------------------------- |
| 操作系统         | Ubuntu 20.04                        |
| CPU              | Intel Core i5 12400F                |
| GPU              | NVIDIA GeForce RTX 3080（12GB显存） |
| 内存             | 32GB                                |
| Python 版本      | 3.7                            |
| 深度学习框架版本 | PaddlePaddle 2.3.0                  |
| CUDA 版本        | 11.6                           |
| cuDNN 版本       | cuDNN 8.4.0                    |
